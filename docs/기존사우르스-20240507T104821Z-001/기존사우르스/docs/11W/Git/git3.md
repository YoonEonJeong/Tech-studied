# Git/Github 

## 
Git을 수행하기 위해서는 다음과 같은 환경을 설정해야 합니다.
1. **Git**, **Git bash** 설치
    - [git](https://git-scm.com/)에서 **git**을 설치합니다, **git**을 설치하면 자동으로 **git bash**또한 설치됩니다.

## 원격 저장소(github)와 로컬 저장소 연동하기
